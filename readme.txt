CY-Meteo

Execution command: "./script.sh"

This program is a meteo data collector which display graphics according to those data. It has been created by Nahil EL BEZZARI and Rayen Mezni. The program requires you to install the latest version of gnuplot as well as all the files you can find in this GitHub. You can add options after the execution command, some are required and other aren't. If you want to know more about it enter the following command: "./script.sh --help"
